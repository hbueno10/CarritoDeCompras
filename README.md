# CarritoDeCompras

Proyecto realizado como parte de la tarea de refactorización en Java.

## ✔ Objetivos cumplidos

- Separación del programa principal y el carrito.
- Eliminación de duplicación de código.
- Manejo de excepciones para ítems nulos o inválidos.
- Encapsulación total de la clase `Item`.
- Validación de precios y cantidades.
- Uso de constantes para evitar “magic numbers”.
- Implementación de `toString()` en `Item`.
- Carrito escalable y fácilmente modificable.
- Documentación completa con JavaDoc.
